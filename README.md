# Bio-Inspired Quadruped Robot

A bio-inspired robotic platform designed to mimic scorpion-like locomotion using a quadruped leg mechanism. This project integrates mechanical design, embedded systems, and intelligent control for adaptive movement.

---

## Features

* Quadruped walking mechanism
* Servo-based joint control
* Embedded control system (ESP32 / Arduino)
* Sensor integration (e.g., BH1750, IMU)
* Future-ready for wireless communication
* Modular design for experimentation

---

## Hardware Components

* ESP32 / Arduino
* PCA9685 Servo Driver
* Servo Motors (MG996R or similar)
* Power Supply (Battery Pack)
* Sensors (BH1750, Ultrasonic, etc.)
* 3D Printed Body Parts

---

## Software Stack

* Arduino IDE / MicroPython
* Embedded C++ / MicroPython
* I2C Communication
* PWM Control for Servos

## How It Works

The robot uses coordinated servo movements to generate a walking gait. Each leg operates with multiple degrees of freedom, allowing stable and efficient motion.
